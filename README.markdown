# 🇺🇸 English

## Overview

Arcade Racing Game is a modern pseudo-3D racing game developed with Python and PyQt6. The project features a responsive user interface, multiple race tracks, AI opponents, nitro boost mechanics, multilingual support, light and dark themes, and an arcade-style racing experience.

## Features

- Pseudo-3D racing engine
- Multiple race tracks
- Multiple car selections
- AI-controlled opponents
- Nitro boost system
- Light and dark themes
- Responsive interface
- Multilingual support
  - English
  - Persian (فارسی)
  - Chinese (中文)
- Race timer and ranking system
- Difficulty levels
  - Easy
  - Medium
  - Hard

## Requirements

- Python 3.10 or newer
- PyQt6

## Installation

### Create Virtual Environment (Optional)

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Install Required Libraries

```bash
pip install PyQt6
```

### Alternative Requirements Command

```bash
pip install -U PyQt6
```

## Run

```bash
python "Arcade Racing Game.py"
```

## Controls

| Action | Key |
